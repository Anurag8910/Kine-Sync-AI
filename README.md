# 🏋️ Kine-Sync-AI

> AI-powered fitness tracking and analytics platform built with the MERN stack.

---

## 🚀 Overview

**Kine-Sync-AI** is a full-stack fitness tracking web application that helps users monitor workouts, track health metrics, and visualize progress through interactive dashboards.

It combines **data tracking + analytics + AI-driven insights** to help users improve their fitness journey.

---

## ✨ Features

* 🔐 User Authentication (Login / Signup)
* 📊 Dynamic Dashboard (real-time updates)
* 🏋️ Workout Tracking
* ⚖️ Weight & Body Metrics Tracking
* 📈 Interactive Charts & Graphs
* 🥗 Nutrition Tracking (Macros)
* 💧 Water & Sleep Monitoring
* 🧠 AI-powered insights (planned / integrated)
* 📅 Workout Calendar
* 🔄 Real-time data sync with backend

---

## 🛠 Tech Stack

### Frontend

* React.js
* Tailwind CSS
* Chart Libraries (Recharts / Chart.js)
* Axios

### Backend

* Node.js
* Express.js

### Database

* PostgreSQL
  
### Other Tools

* JWT Authentication
* REST APIs

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Anurag8910/Kine-Sync-AI.git
cd Kine-Sync-AI
```

---

### 2️⃣ Backend Setup

```bash
cd backend
npm install
```

Create a `.env` file:

```
PORT=5000
DATABASE_URL=your_DATABASE_URL
JWT_SECRET=your_secret_key
```

Run backend:

```bash
npm run dev
```

---

### 3️⃣ Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
